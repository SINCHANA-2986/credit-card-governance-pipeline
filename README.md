# Credit Card Fraud Data Governance Pipeline

## Overview
Built an enterprise-grade data governance pipeline using Python and Google BigQuery over 285K+ financial transactions.

## Features
- Data ingestion into BigQuery
- Data quality scoring
- Duplicate detection
- Schema validation
- Metadata cataloging
- Data lineage tracking
- Automated quality dashboards

## Tech Stack
- Python
- Pandas
- Google BigQuery
- SQL
- Google Cloud

## Dataset
Credit Card Fraud Detection Dataset (Kaggle)

## Results
- 285,807 transactions processed
- 1081 duplicate records identified
- Zero nulls across all columns
- Automated quality checks implemented
